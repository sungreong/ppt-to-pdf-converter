# 폰트 추가 가이드

## 📝 단계별 가이드

### 1단계: 폰트 파일 준비
- `.ttf`, `.otf`, `.ttc` 형식의 폰트 파일을 준비합니다.
- 예: `나눔고딕.ttf`, `Malgun Gothic.ttf` 등

### 2단계: 폰트 파일 복사
Windows에서:
```cmd
copy "C:\Windows\Fonts\malgun.ttf" fonts\
```

또는 파일 탐색기에서:
- `fonts/` 폴더를 열고
- 폰트 파일을 드래그 앤 드롭

### 3단계: 컨테이너 재시작
```bash
docker-compose restart
```

### 4단계: 폰트 확인
브라우저에서:
```
http://localhost:9999/fonts
```

또는 명령어로:
```bash
docker-compose exec ppt-converter fc-list | grep "나눔"
```

## ✅ 예시

### 나눔고딕 추가하기
1. `fonts/` 폴더에 `NanumGothic.ttf` 복사
2. `docker-compose restart` 실행
3. `http://localhost:9999/fonts`에서 확인

### 여러 폰트 한번에 추가
- `fonts/` 폴더에 여러 폰트 파일을 모두 복사
- 한 번만 재시작하면 모든 폰트가 적용됩니다

## ⚠️ 주의사항

1. **컨테이너 재시작 필수**: 폰트를 추가한 후 반드시 재시작해야 합니다.
2. **폰트 라이선스**: 상업적 사용 시 폰트 라이선스를 확인하세요.
3. **파일 형식**: `.ttf`, `.otf` 형식을 권장합니다.

## 🔍 문제 해결

### 폰트가 인식되지 않을 때
```bash
# 컨테이너 내부에서 폰트 캐시 강제 업데이트
docker-compose exec ppt-converter fc-cache -fv
```

### 폰트 목록 전체 확인
```bash
docker-compose exec ppt-converter fc-list
```
